# LoneWolf Fang Free v1.2.0

## 日本語

今回の配布版では、前回配布パッケージ以降の変更を反映し、特に package 品質、update 周辺、GUI の使いやすさを改善しました。

### 主な変更
- bitbank / OKX の spot 取引所サポートを追加
- GitHub Releases の latest 取得まわりを改善
- update dialog の文言と挙動を調整
- manual update retry 周辺を改善
- package / updater まわりの不整合を修正
- GUI の案内表示と安定性を改善
- package / launcher / setup を再ビルドし、配布物を更新

### 注意
- Free はライセンス不要です
- 配布は GitHub Releases 経由です
- 既存環境から更新する場合は、最新の package / setup / launcher を利用してください

---

## English

This package includes the changes made since the previous distribution build, with improvements focused on package quality, update behavior, and general GUI usability.

### Highlights
- Added bitbank / OKX spot exchange support
- Improved GitHub Releases latest-version resolution
- Refined update dialog wording and behavior
- Improved manual update retry flow
- Fixed package / updater related inconsistencies
- Improved GUI guidance and general stability
- Rebuilt package / launcher / setup and refreshed release artifacts

### Notes
- Free does not require a license
- distribution is via GitHub Releases
- when updating an existing environment, use the latest package / setup / launcher
